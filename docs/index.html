Quick Installation

To run the application locally, ensure you have Python 3.8+, Node.js, and FFmpeg installed. Then, follow these steps:

Terminal 
git clone https://github.com/your-username/UniversalMediaDownloader.git
cd UniversalMediaDownloader
python -m venv venv
venv\Scripts\activate  # Or 'source venv/bin/activate' on Mac/Linux
pip install -r requirements.txt

Once installed, just run run.bat (Windows) or ./run.sh (Linux) to start the server!
